# Twitterene for emesene

With Twitterene plugin for emesene you can set your emesene avatar, your nickname and your status with your Twitter avatar, nick and status.

This plugin is very easy to use and allows you to choose what you want take from Twitter: for example if you have a plugin that works on emesene's avatar, you can choose to take from Twitter only the nickname and status. THIS PLUGIN UPDATES YOUR STATUS EVERY 60 SECONDS!!

This plugin uses Tweepy libraries. https://github.com/joshthecoder/tweepy

## Install

To install this plugin you have simply to extract twitterene.zip into youremesenefolder/plugin_base. For example on GNU/Linux systems you can find it at

  /usr/share/emesene/plugin_base

or on windows systems at

  c:\programs\emesene\plugin_base
